# MER0003

## Summary

`MER0003` flags `[OutputCache]` on endpoints that also carry tenant, entitlement, quota, plan, or explicit policy-sensitive metadata.

## Intent

The rule exists because ASP.NET output-cache hits can skip action execution. In Meridian, that can bypass filters such as `ConsumeFeature`, `TenantScoped`, `RequirePlan`, explicit policy checks, and step-up/non-anonymous guards.

## What It Reports

Reports:

```csharp
[HttpGet]
[OutputCache(Duration = 120)]
[ConsumeFeature(SubscriptionConstants.FeatureKeys.GeographicSearch)]
public Task<ActionResult<SearchResponse>> SearchAsync() {
}
```

```csharp
[OutputCache(Duration = 120)]
public sealed class ReportController : BaseApiController {
    [HttpGet]
    [TenantScoped]
    public Task<ActionResult<ReportDto>> GetAsync() {
    }
}
```

## What It Does Not Report

Does not report:

```csharp
[HttpGet]
[OutputCache(Duration = 120)]
public Task<ActionResult<ReferenceDataDto>> GetPublicReferenceAsync() {
}
```

## Examples

Incorrect:

```csharp
[Authorize]
[OutputCache]
[HttpGet("api/reports/{tenantId}")]
public Task<ReportDto> GetReportAsync(string tenantId) => _reader.ReadAsync(tenantId);
```

Correct:

```csharp
[Authorize]
[HttpGet("api/reports/{tenantId}")]
public Task<ReportDto> GetReportAsync(string tenantId) => _reader.ReadAsync(tenantId);
```

### Entitlement-Gated Endpoint

Incorrect:

```csharp
[HttpGet("api/search")]
[OutputCache(Duration = 60)]
[ConsumeFeature(SubscriptionConstants.FeatureKeys.GeographicSearch)]
public Task<SearchResponse> SearchAsync(SearchRequest request) {
    return _service.SearchAsync(request);
}
```

Correct:

```csharp
[HttpGet("api/search")]
[ConsumeFeature(SubscriptionConstants.FeatureKeys.GeographicSearch)]
public Task<SearchResponse> SearchAsync(SearchRequest request) {
    return _service.SearchAsync(request);
}
```

### Explicit No-Store

Incorrect:

```csharp
[HttpGet("api/subscriptions/current")]
[OutputCache(Duration = 300)]
[TenantScoped]
public Task<MySubscriptionDto> GetCurrentAsync() {
    return _service.GetCurrentAsync();
}
```

Correct:

```csharp
[HttpGet("api/subscriptions/current")]
[ResponseCache(Location = ResponseCacheLocation.None, NoStore = true)]
[TenantScoped]
public Task<MySubscriptionDto> GetCurrentAsync() {
    return _service.GetCurrentAsync();
}
```

## Current Rollout

- Live warning scope: `warning` in analyzer-scope backend projects during live analysis and explicit analyzer-enabled lanes
- Inventory command: `rtk pnpm backend:analyzers:inventory -- --diagnostics MER0003 --project apps/backend/Meridian.API/Meridian.API.csproj`
- Fallback loading check: `rtk pnpm backend:analyzers:validate:build -- --diagnostics MER0003`
- Candidate artifact: [Backend custom analyzer staged inventory rollout](docs/analysis/2026-05/2026-05-07-backend-custom-analyzer-staged-inventory.md)

`MER0003` is warning-enabled in committed `.editorconfig` for analyzer-scope live analysis and explicit analyzer-enabled lanes. Normal backend builds still keep analyzers off by default through `apps/backend/Directory.Build.props`, so use `backend:analyzers:validate`, `backend:analyzers:inventory`, or `backend:analyzers:validate:build` when you need replayable command-line evidence.

## Preferred Refactor

1. Remove `[OutputCache]` from tenant, entitlement, quota, plan, or policy-sensitive endpoints.
2. Prefer `[ResponseCache(Location = ResponseCacheLocation.None, NoStore = true)]` when the endpoint must explicitly disable shared caching.
3. Introduce `[OutputCache]` only with a reviewed persona-safe cache policy.

## Edge Cases

- Class-level `[OutputCache]` reports once when the controller class or any action adds sensitive metadata.
- Method-level `[OutputCache]` reports when the controller class carries sensitive metadata.
- Public/static reference endpoints without sensitive metadata do not report.

## Suppression Guidance

Default stance: remove or replace `[OutputCache]`.

A suppression is only justified when a reviewed policy proves the cached payload is persona-safe and cannot skip quota, entitlement, tenant, plan, or policy behavior.
