# MER0012

## Summary

`MER0012` flags source `Microsoft.Extensions.Diagnostics.HealthChecks.IHealthCheck` implementations without a matching generic health-check registration.

## Intent

Unregistered health checks are dead operational code. Health-check implementations should be visible in registration and then verified by runtime health endpoint tests.

## What It Reports

Reports non-abstract source classes implementing `Microsoft.Extensions.Diagnostics.HealthChecks.IHealthCheck` when they are either API-host-local or non-public library checks, and the same compilation does not register them through:

- `AddCheck<THealthCheck>(...)`
- `AddTypeActivatedCheck<THealthCheck>(...)`

## Examples

Incorrect:

```csharp
public sealed class SearchHealthCheck : IHealthCheck {
    public Task<HealthCheckResult> CheckHealthAsync(HealthCheckContext context, CancellationToken cancellationToken = default) {
        return Task.FromResult(HealthCheckResult.Healthy());
    }
}
```

Correct:

```csharp
services.AddHealthChecks()
    .AddCheck<SearchHealthCheck>("search", tags: new[] { "search", "dependencies" });
```

### Type-Activated Registration

Incorrect:

```csharp
public sealed class StorageHealthCheck : IHealthCheck {
    public Task<HealthCheckResult> CheckHealthAsync(HealthCheckContext context, CancellationToken cancellationToken) {
        return _storage.PingAsync(cancellationToken);
    }
}
```

Correct:

```csharp
services.AddHealthChecks()
    .AddTypeActivatedCheck<StorageHealthCheck>(
        "storage",
        failureStatus: HealthStatus.Unhealthy,
        tags: new[] { "storage", "dependencies" },
        args: new object[] { options.BucketName });
```

### Feature Extension Registration

Incorrect:

```csharp
public sealed class ReportsHealthCheck : IHealthCheck {
    public Task<HealthCheckResult> CheckHealthAsync(HealthCheckContext context, CancellationToken cancellationToken) {
        return _reports.CheckAsync(cancellationToken);
    }
}
```

Correct:

```csharp
public static IServiceCollection AddReportHealthChecks(this IServiceCollection services) {
    services.AddHealthChecks()
        .AddCheck<ReportsHealthCheck>("reports", tags: new[] { "reports" });

    return services;
}
```

## Preferred Refactor

Register each health check from the feature or hosting registration surface, then keep endpoint-level behavior covered by tests.
