# MER0011

## Summary

`MER0011` flags static mutable state in controllers and authentication/session handlers.

## Intent

Static mutable caches, dictionaries, timers, and throttle windows hide lifecycle, eviction, tenant scope, and test isolation. Runtime state should be owned by an injectable service with explicit bounds.

## What It Reports

Reports static fields with mutable collection/cache/timer types in:

- controller classes
- `*AuthenticationHandler`
- `*AuthHandler`
- `*SessionHandler`

## Examples

Incorrect:

```csharp
public sealed class SessionController : ControllerBase {
    private static readonly Dictionary<string, string> Sessions = new();
}
```

Correct:

```csharp
public sealed class SessionController : ControllerBase {
    private readonly ISessionStore _sessions;

    public SessionController(ISessionStore sessions) {
        _sessions = sessions;
    }
}
```

### Static Cache in Controller

Incorrect:

```csharp
public sealed class TokenController : ControllerBase {
    private static readonly MemoryCache TokenCache = new(new MemoryCacheOptions());
}
```

Correct:

```csharp
public sealed class TokenController : ControllerBase {
    private readonly ITokenCache _tokenCache;

    public TokenController(ITokenCache tokenCache) {
        _tokenCache = tokenCache;
    }
}
```

### Static State in Auth Handler

Incorrect:

```csharp
public sealed class ApiKeyAuthenticationHandler : AuthenticationHandler<AuthenticationSchemeOptions> {
    private static readonly ConcurrentDictionary<string, DateTimeOffset> SeenKeys = new();
}
```

Correct:

```csharp
public sealed class ApiKeyAuthenticationHandler : AuthenticationHandler<AuthenticationSchemeOptions> {
    private readonly IApiKeyReplayStore _replayStore;

    public ApiKeyAuthenticationHandler(IApiKeyReplayStore replayStore) {
        _replayStore = replayStore;
    }
}
```
