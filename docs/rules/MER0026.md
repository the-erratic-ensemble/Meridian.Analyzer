# MER0026

## Summary

`MER0026` flags deeply nested ternary chains.

## Intent

Long ternary chains bury decision trees in one expression. They are hard to scan, hard to diff, and hard to adjust safely when the ranking or classification rules change.

## What It Reports

Reports:

```csharp
return isExactRail
    ? 0
    : hasRailName
        ? 1
        : hasTrainName
            ? 2
            : looksLikeStation
                ? 3
                : 99;
```

```csharp
return row.TransportModeCode == "TRA"
    ? 0
    : row.TransportModeCode == "RAIL"
        ? 1
        : row.StopTypeCode == "RAI"
            ? 2
            : row.StopTypeCode == "RSE"
                ? 3
                : 99;
```

## What It Does Not Report

Does not report:

```csharp
return isExactRail ? 0 : hasRailName ? 1 : 99;
```

```csharp
if (isExactRail) {
    return 0;
}

if (hasRailName) {
    return 1;
}

if (hasTrainName) {
    return 2;
}

return looksLikeStation ? 3 : 99;
```

```csharp
return isExactRail ? 0 : 99;
```

## Examples

Incorrect:

```csharp
return candidate.IsPriorityA
    ? 0
    : candidate.IsPriorityB
        ? 1
        : candidate.IsPriorityC
            ? 2
            : candidate.IsPriorityD
                ? 3
                : 99;
```

Correct:

```csharp
if (candidate.IsPriorityA) {
    return 0;
}

if (candidate.IsPriorityB) {
    return 1;
}

if (candidate.IsPriorityC) {
    return 2;
}

return candidate.IsPriorityD ? 3 : 99;
```

## Preferred Refactor

1. Replace long ternary chains with named locals, guards, or helper methods.
2. Keep the final expression short enough to read in one pass.
3. When the logic is ranking or classification logic, give each decision step a name.

## Edge Cases

- Generated code does not report.
- A two-branch nested conditional such as `a ? 0 : b ? 1 : 99` does not report.
- The rule reports only the root conditional in the chain.

## Suppression Guidance

Refactor first. Suppress only when a longer nested ternary is genuinely clearer than the available statement or helper-based alternatives.
